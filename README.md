# **FinVisionX — CAPM & Quantitative Market Analysis (Indian Equity Market)**

**FinVisionX** is an advanced, interactive financial analytics dashboard designed for the **Indian stock market**, integrating **CAPM modeling, portfolio optimization, statistical forecasting, and quantitative risk analysis** in one unified platform. Built using **Streamlit**, **Machine Learning models**, and modern quantitative finance techniques, FinVisionX enables investors, researchers, and students to analyze stock performance with professional-grade tools.

##  **Key Features**

### **Advanced CAPM Analysis**

* Regression-based Beta estimation
* Alpha evaluation
* R² goodness of fit
* Expected return modeling

### **Price Analytics**

* Interactive price charts
* Technical indicators (MA50, MA200, RSI, Bollinger Bands)
* Asset comparison

### **Risk & Rolling Metrics**

* Rolling Beta
* Rolling annualized volatility
* Multi-asset comparison

### **Portfolio Optimization (Efficient Frontier)**

* Monte Carlo simulation (1,000–20,000 simulations)
* Maximum Sharpe portfolio identification
* Optimal asset weight allocation
* Risk–Return visualization cloud

### **Return Forecasting**

* Predictive modeling using **ensemble learning (Random Forest)**
* Lag-based feature engineering
* Confidence interval bands
* Trend visualization

### **Correlation & Clustering**

* Clustered correlation heatmap
* Diversification insights
* Hierarchical clustering for asset grouping
